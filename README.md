# Renovera Projetos Eletricos

Landing page publica da Renovera para projetos eletricos, subestacoes, protecao, entrada de energia e suporte tecnico comercial.

## Tecnologias

- React
- TypeScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicacao

Este repositorio publica automaticamente no GitHub Pages pelo workflow em `.github/workflows/deploy-pages.yml`.

URL publica:

```txt
https://renovera1.github.io/renovera-projetos-eletricos/
```
